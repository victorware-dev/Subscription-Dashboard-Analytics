4. Rest Operator

Calculate multiple expenses

function calculateExpenses(...amounts){
   return amounts.reduce(
      (sum,val)=>sum+val,
      0
   );
}
5. map()

Create UI Cards

subscriptions.map(subscription => ...)
6. filter()

Active subscriptions

subscriptions.filter(
   sub => sub.active
)
7. reduce()

Total monthly spend

subscriptions.reduce(
   (sum, sub) =>
      sum + sub.monthlyCost,
   0
);
